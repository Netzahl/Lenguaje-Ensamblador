# Lenguaje Ensamblador

Ejercicios y prácticas de Lenguaje Ensamblador usando TASM.

## Herramientas necesarias
- TASM (Turbo Assembler)
- TLINK (Turbo Linker)

## Compilación
```bash
tasm archivo.asm
tlink archivo.obj
archivo.exe
```

**Nota:** Los archivos TASM.EXE y TLINK.EXE no están incluidos en este repositorio.
